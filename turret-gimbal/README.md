# Paintball Sentry Turret — Pan/Tilt Gimbal

DIY NEMA 23 closed-loop pan/tilt gimbal for an autonomous paintball (non-lethal) sentry turret.
Bolt-together, zero-machining build: flat DXF plates cut by SendCutSend + bolt-on catalog parts.

**Start with `CLAUDE.md`** — it has the full project state (safety posture, architecture, design
decisions, confirmed parts, open items, and the DXF status).

```
CLAUDE.md     project context (read first)
bom/          live BOM + parts lists
cad/dxf/      6 plate cut files (6061-T6, 6 mm, mm units)
docs/         build plan, assembly guide, drawings, specs (md + pdf)
datasheets/   confirmed-part datasheets + INDEX.md
```
