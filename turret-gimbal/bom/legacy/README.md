# Legacy BOM spreadsheets (superseded)

These are **old, no-longer-maintained** parts-list spreadsheets, kept only for
history. They predate the integrated BOM.

**The live BOM is the Markdown at the repo root:** `Turret_BOM.md` (table) and
`Turret_BOM_checklist.md` (Ordered/Received checklist). Do not edit the files
here expecting them to flow anywhere.

- `Gimbal_BOM.xlsx` — early gimbal-only parts list
- `RPi_Turret_Parts_List.xlsx` — early electronics/compute list
- `Turret_Full_Parts_List.xlsx` — early combined list
