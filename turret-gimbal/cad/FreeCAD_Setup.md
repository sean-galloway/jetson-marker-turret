# FreeCAD setup & asset download checklist

Everything needed to assemble the turret in FreeCAD. Use **FreeCAD 1.0+** (built-in
Assembly workbench; fixes the old topological-naming issues).

Drop downloaded models into `turret-gimbal/cad/step/`. Plate solids generated by
`tools/freecad_build.py` go in `cad/step/plates/`.

---

## 1. Software to install
| Tool | URL | Why |
|------|-----|-----|
| **FreeCAD 1.0+** | https://www.freecad.org/downloads.php | Modeling + built-in **Assembly** + **TechDraw** |
| **Fasteners** workbench | FreeCAD → Tools → **Addon Manager** → "Fasteners" | Generate ISO/DIN screws, nuts, standoffs, dowels on the fly (no STEP needed) |
| **OpenSCAD** | https://openscad.org/downloads.html | Render the parametric parts I wrote (`cad/scad/*.scad`) → export STL/STEP |

Accounts you'll likely need for some downloads:
- **GrabCAD** (free): https://grabcad.com/library — community models (Pi, Jetson, switch).
- **NVIDIA Developer** (free): https://developer.nvidia.com — Jetson design resources.

## 6. Model from dimensions (no good STEP — build it yourself)
- **Lazy-susan bearing** (use measured dims, §2).
- **Rack + GeeekPi parts** (DeskPi RackMate T2, 10" PDU, 12-port patch panel, switch mount, 6.91" displays) — grab dimensions from the product pages / deskpi.com & 52pi.com and model simple blocks; the rack mostly sets the shelf interface.
- **TP-Link TL-SG108E** — GrabCAD or a dimensioned block.
- **RadioMaster TX16S / ELRS RX** — not needed for the turret CAD.
- **Screws / nuts / standoffs / dowels** — generate with the **Fasteners workbench** instead of downloading.

---

## 7. Suggested order of work in FreeCAD
1. Run `freecad_build.py` → 6 plate solids.
2. Import the gimbal-core STEPs (motor, gearbox, bearings, hub, shoulder screw).
3. Model the lazy-susan from dims.
4. Assemble pan + tilt (Assembly WB); add fasteners (Fasteners WB).
5. Add the payload sensors once the **marker scan** lands (drives the shroud + frame).
6. TechDraw pages for per-step build drawings; exploded view for the blow-up.
